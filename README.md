# (Vue - Vuetify - TS) Boilerplate

## Folder Structure
```
src
  - assets
  - components
  - layouts
  - views
  - helpers
    |- axios
    |- importer
  - plugins
    |- vuetify
  - router
    |- routes
  - store
    |- modules
      |- auth

tests
  - unit

```


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files (Javascript Standard)
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```
